Exploración de los datos

Después de la realización de la descripción de los datos, procederemos a realizar la respectiva exploración de los mismos. Dentro de esta exploración aplicaremos pruebas estadísticas básicas que nos permitan descubrir propiedades en los datos que difícilmente se verían a simple vista, y crear gráficos de frecuencia, de línea y de distribución de los datos. Este informe nos permitirá esencialmente determinar qué tan consistentes y completos se encuentran los datos recolectados.

Correlación entre los atributos del conjunto de datos

Para la realización del análisis o exploración de los atributos que conforman el conjunto de datos, comenzamos analizando la correlación que existe entre los distintos atributos numéricos del conjunto de datos por medio de los coeficientes de Pearson, Spearman y Kendall. Esto nos permitirá determinar el grado de asociación entre los atributos del conjunto de datos, para luego seleccionar aquellos que presenten una mayor asociación o correlación. Para los coeficientes de Spearman y Kendall obtuvimos cierta similitud en los valores. Estos valores muestran una fuerte correlación entre los atributos Área Cosechada y Área Sembrada, lo que posiblemente indica que entre mayor sea la superficie de cultivo de café sembrado mayor será la superficie que se cosechará. Esta fuerte correlación entre los atributos Área Cosechada y Área Sembrada nos permite tenerlos en cuenta para la posterior selección de atributos en la fase de preparación de los datos. Los valores obtenidos por los coeficientes de Spearman y Kendall también muestran una fuerte correlación entre el atributo Producción y los dos anteriores (Área Cosechada y Área Sembrada), lo que también indica que la producción de café en un cultivo depende del área sembrada, es decir, entre mayor sea el área sembrada mayor será la producción de café. El atributo Producción al indicar una fuerte correlación con los dos atributos anteriores (Área Cosechada y Área Sembrada), también se convierte en un candidato para ser seleccionado en etapas posteriores. También podemos notar una correlación débil entre el atributo Rendimiento y el atributo Producción; esta correlación débil también se presenta entre el atributo Rendimiento y, los atributos Área Cosechada y Área Sembrada. También podemos mencionar una correlación muy débil entre el atributo Año y el atributo Rendimiento.

Esta correlación podría indicar que el rendimiento de un cultivo de café es dependiente en cierta medida de la producción, el área cosechada, el área sembrada, como también el año de la siembra. También podemos suponer que la razón de una correlación tan débil entre el Rendimiento con los otros atributos es debido a la presencia de datos atípicos o también por datos faltantes.

Esta correlación entre las variables también la podemos observar más claramente en el diagrama de correlación de Pearson en el boton de abajo. En este gráfico, al igual que con las matrices de Kendall y Spearman (aunque con una pequeña variación en los resultados), podemos ver que existe una correlación lineal fuerte entre el área cosechada y el área sembrada. Esta fuerte correlación, como ya mencionamos, nos dice que a mayor área sembrada mayor será el área cosechada. Por otro lado, también observamos una relación fuerte entre la producción, el área cosechada y el área sembrada, indicando esto que la producción es linealmente dependiente del área sembrada, es decir, a mayor área sembrada mayor será la producción. Como también podemos observar en la matriz de correlación de Pearson, existe una correlación débil entre la producción y el rendimiento, indicando nuevamente una posible dependencia entre los dos atributos. Como ya mencionamos, la correlación débil entre estos dos atributos podría estar indicándonos la existencia de datos faltantes en el conjunto de datos, como también la presencia de datos atípicos.

Matriz de dispersión

Para continuar con el análisis del conjunto de datos, presentaremos una matriz de dispersión de las variables numéricas que conforman el conjunto de datos para visualizar la correlación entre las variables de una forma mucho más visual. En el gráfico 5 a simple vista podemos identificar que existe una correlación lineal entre el área cosechada y el área sembrada, como también entre la producción, el área sembrada y el área cosechada. Podemos deducir de una forma superficial que a medida que aumenta el área sembrada también aumentará el área cosechada. Este efecto es igual para la producción, es decir, la producción aumentará a medida que el área cosechada y el área sembrada aumente. Esta fuerte correlación lineal entre los atributos producción, el área sembrada y el área cosechada, nos permite identificarlos como candidatos para una futura selección de atributos en la etapa de preparación de los datos. Para el atributo Rendimiento la relación existente con los demás atributos es difícil de identificar a simple vista, pero si observamos con detenimiento podemos notar cierta relación lineal con los atributos producción, área cosechada y área sembrada.
