Área Sembrada

Cómo podemos observar en la gráfica 38, la cantidad de datos atípicos para el atributo área sembrada en el conjunto de datos es de 687, prácticamente la misma cantidad que para el atributo área cosechada, es decir, un 9.5% de los valores para este atributo tienen un comportamiento que se desvía del resto de los datos. Dado que el total de datos atípicos para el atributo área sembrada también es un número considerablemente grande, al igual que para el atributo área cosechada, serán reemplazados por la media aritmética del atributo para que estos valores más adelante no afecten negativamente la construcción del modelo en la etapa de modelado. Cabe mencionar que la similitud en cuanto a la cantidad de datos atípicos entre el atributo área cosechada y el atributo área sembrada podría deberse a que estos datos están directamente relacionados, es decir, probablemente estos datos se deben a comportamientos reales de los cultivos de café pero que se salen de la norma y no son originados por errores de registro en el conjunto de datos.
